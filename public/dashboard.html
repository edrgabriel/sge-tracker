Cadastro de Equipamentos

Utilize este formato quando o laboratório enviar novos equipamentos para abastecer a base (Status Disponível).

numero_internoserial
10245SN12345678
10246SN12345679
  • numero_interno: Máximo de 5 dígitos!
  • serial: Obrigatório, campo alfanumérico.

Distribuição em Massa

Para envio da sede aos porta-malas. Aloca os ativos ao respectivo técnico lendo pelo Serial escolhido.

numero_internoserialnome_tecnico
10245SN12345678Paulo Silva
  • numero_interno/serial: Preencher ao menos um!
  • nome_tecnico: Aceita nome limpo (ex: Paulo Silva) ou a exata ID numérica correspondente.

Registro de Serviços Concluídos

Usado no Fim do Dia para baixar todas O.S. de uma vez. O STOKI deduz sozínho os técnicos a partir do serial.

serialtipo_servicodata
SN12345678Instalação2026-04-18
  • serial: Apenas ele já acha o técnico!
  • tipo_servico: Deve bater exatamente com a nomenclatura das configurações (ex: Manutenção).

  • sub_cidades: Preencha separando cada tag de cidade sempre acompanhando uma vírgula entre elas.
